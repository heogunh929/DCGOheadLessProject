/goal 최종 목표는 고정된 https://github.com/DCGO2/DCGO 원본 snapshot의 전체 카드풀과 battle rule을 headless RL.Engine으로 최대한 같은 의미와 추적 가능한 구조로 이식한 뒤, 그 전체 카드풀 환경에서 self-play 모델을 학습하는 것이다. ST1~ST3는 공통 엔진 구조를 검증하는 중간 vertical slice이며 ST1~ST3 gate 통과를 RL 학습 진입 근거로 사용하지 않는다. 먼저 현재 engine-parity queue의 core timing, selection pause/resume, security, MultipleSkills, counter/block, trace parity를 완료한다. 그 다음 DCGO2/DCGO의 정확한 commit SHA를 사용자 승인으로 고정하고, 해당 snapshot의 모든 CardBaseEntity asset, CardIndex, CardId, variant, CardEffectClassName, CardEffect source, keyword, complex mechanic을 inventory한다. 모든 asset identity는 CardId만으로 평탄화하지 않는다. 모든 카드 또는 source effect에는 원본 경로를 추적할 수 있는 대응 파일/record가 있어야 하며 Catalog는 registry 역할만 담당한다. 원본 의미를 알 수 없는 효과, missing source body, Unsupported, PartiallyImplemented, StubbedForValidation, UnknownVariant는 silent no-op이나 추측 구현으로 숨기지 않는다. 직접 zone list 수정과 core service의 카드 ID shortcut을 금지한다. DCGO/Assets/Scripts 원본은 읽기 전용 Source of Truth로 유지한다. Codex는 commit 또는 push하지 않고 추천 commit message만 보고한다. Full DCGO Snapshot Completion Gate와 사용자 승인이 모두 완료되기 전에는 ObservationEncoder, RewardCalculator, DatasetExporter, Trainer, RL Environment API, 학습용 self-play dataset을 구현하지 않는다. 모든 분석, 진행, 테스트, TODO는 한국어로 작성하고 한 번에 queue 항목 하나만 수행한다.
